Folder code

create_unique_df.py - In the database, there are some questions that repeat but have different answers.
                      This script creates a DataFrame with unique question values and combines all the
                      answers related to the same question.

preprocessing.py - Contains a function for removing stop words and characters, as well as functions for stemming,
                   lemmatization, and functions related to TF-IDF.

main.py - This script contains the complete code for running the model with the best parameters.
